# 프로젝트 이름

사칙연산을 수행하는 계산기 프로젝트

## 소개
이 프로젝트는 자바의 기초부터 객체지향, 컬렉션, Enum, 람다 & 스트림까지 한 단계씩 성장하며 구현하는 계산기 프로젝트입니다.

사용자로부터 2개의 숫자와 사칙연산자 문자를 입력받을 수 있다.
프로그램은 예외를 처리와 계산을 하여 연산 결과를 반환한다.
계산기는 계속, 수정, 삭제, 종료 기능이 있다.

## 사용 기술
- Java 17
- IDE: IntelliJ
